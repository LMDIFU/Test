# Simple Telegram Bot

## Usage

1. Install dependencies:
   ```bash
   pip install python-telegram-bot
   ```
2. Run the bot:
   ```bash
   python simple_bot.py
   ```

Before running, make sure to set your bot token in `simple_bot.py`.
